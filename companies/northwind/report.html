The performance score

Our performance score measures what matters.

Most industry tools focus on mention rate, which doesn't measure whether a response was accurate or favorable.

We score the quality of each LLM response using a reproducible method that allows for immediate interpretation and trending.

How the 0 to 5 score is computed
  1. Every prompt gets success criteria, drafted by a human
  2. An agent grades each response against that criteria
  3. Each criterion gets pass (1), partial (0.5), or fail (0)
  4. The score is the weighted share of criteria passed
